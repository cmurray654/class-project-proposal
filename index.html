Application Overview

SmartStudy is a simple study planner that helps students organize assignments, track deadlines, and stay productive. From the user’s point of view, it is an easy-to-use web app that allows task creation, reminders, and progress tracking.

Technology Overview

  • Languages: HTML, CSS, JavaScript
  • Libraries: None (vanilla JavaScript)
  • Platform: GitHub for version control
  • Hosting: GitHub Pages

Proposal Last Updated

Last updated: October 22, 2025

Feature List (Prioritized)

  1. Create, edit, and delete tasks
  2. Set reminders for assignments
  3. Sort tasks by subject or due date
  4. Progress tracking dashboard

Future Feature List

  • Calendar view integration
  • Mobile app version
  • Study group collaboration
  • Voice command task input

Basic Technical Features

  • Uses HTML, CSS, and JavaScript
  • Responsive layout for different devices
  • LocalStorage to save tasks
  • Simple DOM manipulation for adding tasks

Advanced Technical Features

  • Optional API connection for login and sync
  • Progress chart built with Canvas API
  • Dark mode using CSS variables
  • Offline mode using Service Worker
